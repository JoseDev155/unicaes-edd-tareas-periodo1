# Parcial I - EDD

## Dependencias

Instalar la dependencia `prompt-sync`.

Se puede hacer ejecutando:

```bash
npm i prompt-sync
```

O usando el `package.json` y `package-lock.json`:

```bash
npm i
```

**Nota**: `prompt-sync` no permite ingresar tildes.

## Ejecutar archivos con Node.js

Comando:

```bash
node <archivo-js>.js
```
